Projects

ADS-B Receiver Handling

I designed custom software to interpret incoming ADS-B signals from airplanes and send them to a web server with a Raspberry Pi, SDR dongle, and 1090 MHz antenna. I produced a far cheaper ADS-B receiver setup to be built and installed in bulk for airports. Finally, I wrote an installation guide and further documentation for users.

Game Development

Game development in Unity (C#) and Godot (C#, GDScript) has been an ongoing passion project that allows me to blend my artistic creativity with a love of math and programming. I have developed several small scale proof of concepts games.

Verilog Single Cycle Processor

Implemented components of a MIPS processor in Verilog and formed them together to simulate a CPU running ARMv8 instructions. Included process controler, ALU, sign extension, memory, and register files.
